1. В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.
на базе http://try.redis.io/

предположим хранение осуществляется следующим образом:
SADD <ipAddress> <visit_timestamp_>
таким образом можно просто вычислить количество значений по ключу с определенным ip адресом.
SCARD <ipAddress>
Но для меня остается неясным вопрос, как решить задачу серых ip, за которыми находятся множества клиентов. я так понимаю тут необходимо добавлять данные о приложении, с которого происходит обращение к ресурсу


2. При помощи базы данных Redis решите задачу поиска имени пользователя по электронному адресу и наоборот, поиск электронного адреса пользователя по его имени.

SADD userName username@gmail.com username@rambler.ru username@mail.ru
Set username@gmail.com userName
Set username@rambler.ru userName
Set username@mail.ru userName

соответственно:
SMEMBERS userName - получить все почтовые ящики
get username@gmail.com - получить имя пользователя
